# 人事管理系统 java1153

## 项目概述

人事管理系统是一款基于 SpringBoot 和 Vue.js 构建的现代化企业人事管理工具。

## 技术栈

### 前端

- Vue.js：构建动态交互界面

### 后端

- SpringBoot：提供高效稳定的开发框架
- Mybatis：数据持久化映射框架

### 数据库

- MySQL 5.x/8.x：双版本适配

### 开发环境

- JDK8：Java 开发工具包
- Maven：项目管理工具

## 功能模块

- 用户管理
- 员工信息管理
- 考勤管理
- 薪资管理
- 权限管理
- 报表统计

## 系统角色

- 系统管理员
- 人事管理员
- 员工

## 运行环境

- 操作系统：Windows/Linux/MacOS
- 数据库：MySQL 5.x/8.x
- 依赖管理：JDK8+Maven

## 部署步骤

1. 安装 MySQL 数据库
2. 部署后端服务
3. 部署前端服务
4. 启动应用
